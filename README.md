# Bank Customer Churn Prediction

SkillFactory ML educational project focused on predicting bank customer churn.

## Project Description

The project includes data preprocessing, exploratory data analysis, model training and model evaluation.

Models used:
- Logistic Regression
- Logistic Regression with Polynomial Features
- Decision Tree
- Random Forest

Project notes and conclusions inside the notebook are written in Russian.

## Files

- `Project3.ipynb` — main notebook
- `data/churn.csv` — dataset
- `requirements.txt` — required Python libraries
